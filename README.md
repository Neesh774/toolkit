## Development

Clone the project

```bash
  git clone https://github.com/coders-lounge/bot.git
```

Go to the project directory

```bash
  cd bot
```

Install dependencies

```bash
  yarn install
  # or
  npm install
```

Start the bot

```bash
  yarn dev
  # or
  npm run dev
```
